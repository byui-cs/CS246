Objectives

  • Submit your Final Project Report.

Final Project Report

This week, you'll submit a report dicussing how the project development process went. Some people call this a postmortem.

Instructions

  1. Read the Project Report Instructions.

  2. Review the project rubric, which you'll use to evaluate your project.

  3. Use this project template to write your report.

    Most of the report can be identical for everyone on your team. Only the Time Breakdown, Project Evaluation, Grade Calculation, and Team Member Contribution sections should be individual (and confidential).

  4. Upload the final report to I-Learn. Everyone on the team must submit their own report.
