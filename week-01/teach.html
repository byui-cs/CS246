Write a Java Hello World program

With your system configured, you're ready to write your first Java program. Here are the guidelines to follow:

  1. Make sure you review the Google Java Style Guide to see how Java Guide should be styled for this course.

  2. Create a file called HelloWorld.java

  3. In that file, create a public Java class called HelloWorld

  4. Add the main() method, making sure that you write its signature correctly (review the preparation reading if necessary).

  5. Add another method to the the HelloWorld class called sayHello(). This method should print "Hello World" to the console.

  6. Have main() create a new instance of the HelloWorld class and call its sayHello() method. This is all that main() should do.

  7. Compile your code:

    javac HelloWorld.java

  8. Execute your code:

    java HelloWorld

Once you've finished, or if you get stuck, compare your code to the teacher's solution.

Now that you can write simple Java programs, it's time to make something useful. Write a program that asks the user for the radius of a circle, then have it print the area and circumference rounded to two decimal places.

  1. Before you begin, you should read the following:

    • Formatting Numeric Output
    • How to use Java's Scanner class to get user input.

  2. Create a public class called CircleCalculator and save it in a .java file with the same name.

  3. Add a function to the class called getRadius() that gets the radius from the user as a float.

  4. Add a function to the class called displayCircumference() that takes the radius as a parameter, calculates the circumference, and displays it to two decimal places.

  5. Add a function to the class called displayArea() that takes the radius as a parameter, calculates the circumference, and displays it to two decimal places.

  6. As before, have main() create an instance of the class. Then call the instance's getRadius(), displayCircumference(), and displayArea() methods.

Congratulations! You now know how to write, compile, and run basic Java programs.
